研究資料寄存所更新：當前版本重大更新_v6.7.0

2024-08-30

depositar 在 7 月 18 日進行了當前版本的重大更新。本頁將介紹與使用者有關的重要更新內容。

重要更新內容

使用者可透過「使用者名稱」或「電子郵件」進行登入

在過去，使用者僅能透過「使用者名稱」進行登入，如今該限制已放寬。

舊版 API Key 已失效，API 須改用 API Token 進行授權驗證

depositar 的部分 API 功能僅授權給已註冊的使用者，在過去，每位使用者都配有一組持續有效的 API Key，使用者可透過該 API Key 進行驗證。但僅有一組的 API Key，不僅存在安全上的疑慮，也不利於 API 使用上的管理。本次更新後，depositar 已不再支援 API Key 的驗證方式，取而代之，使用者可透過個人資料頁面上的「API Tokens」頁籤，申請不限數量的 API Tokens， 並能隨時撤銷任一 Token。

資源新增「Table」檢視

針對表格資料（如 csv, xls 等），資料上傳者如今可在資源的「檢視」選項中，新增「Table」的檢視模式。相較「資料瀏覽器」的模式，「Table」檢視的主要特色為：

  • 可下載過濾結果為 CSV、TSV、JSON、XML 格式
  • 儲存過濾狀態並產生分享連結
  • 選擇要顯示的欄位

值得注意的是，「Table」檢視目前並不支援中文搜尋、亦無地圖瀏覽、製作簡易圖表等「資料瀏覽器」所固有的功能。有關「Table」檢視更進一步的說明，可見 depositar 資料預覽功能介紹。

補充資訊

本次更新基於 CKAN 2.10.4，更新前的 CKAN 版本為 2.9.11。欲了解兩版本間的完整差異，可於 CKAN 官網查看上方兩版本號間的各版本異動說明。
